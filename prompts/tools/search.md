## Description

Search the web for a given query. Returns a list of titles, URLs, and short snippets. Note: Snippets are not full page content; use read_page or read_pages to get the full text from a URL.

## Examples

```json
{"tool": "search", "query": "latest news"}
{"tool": "read_page", "url": "https://example.com/article", "topic": "key points"}
```
